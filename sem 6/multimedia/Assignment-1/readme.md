Steps :  

1. Open : file Open the file KEYBOARD.PSD file in Photoshop 
2. Make layer visible : Make the bearing and gauge layers visible by clicking to the left of the layer icons. 
3. Open file : Open the CLOCK.PSD file in Photoshop 
4. Move : Use the Move tool to drag the clock image and place on top of the keyboard image. 
5. Close file : Close the CLOCK.PSD file. 
6. Rename layer : Right-click on the new Layer-1 and from Layer Properties rename the layer to Clock.   
7. Reorder layer : In the Layers palette drag the Clock layer to the top 
8. Select (MW), Delete : Keeping the Clock layer selected, use the Magic Wand tool to select the white background portion of the clock and press DEL key to make its background transparent. Press CTRL+D to deselect 
9. Change blending mode, Change Opacity : In the Layers palette change the blending mode of the Clock layer to Screen and make its opacity 80%. 
10. Move : Reposition the clock and bearing by dragging them to the right of the image using the Move tool 
11. Transform : Use the Transform function (Edit > Free Transform) to rotate the clock and scale down its size. Press SHIFT while scaling to ensure uniform scaling 
12. Create layer : Create a new layer just above the Background layer by selecting the background layer and clicking on the Create New Layer button at the bottom of the Layers palette. 
13. Rename layer : Rename the new layer to Gradient 
14. Create new gradient : Selecting the Gradient layer open the gradient editor by clicking on the Gradient tool. In the Gradient Editor click to select the red-green preset gradient. Click below the middle of the red green gradient bar to create a new color stop. From the color panel select a yellow color. Click on the New button to create a preset copy of the new gradient. Click OK to close the gradient editor 
15. Apply gradient : Select the Gradient layer in the Layers palette and using the gradient tool drag from top left corner of the image to the bottom right corner. 
16. Change opacity : Decrease opacity of gradient layer to 60% so that the keyboard is visible from below 
17. Save file : Save as TASK3.PSD and TASK3.JPG
