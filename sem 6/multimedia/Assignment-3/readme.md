**EXERCISE – 3 : WORKING WITH SELECTION TOOLS-II** 

Task : Separate the shapes from the background and move each to a different layer 

Steps: 

1. Open file :: Open file ICONS.PSD in Photoshop 
2. Selection: star :: Use Polygonal Lasso for the white star 
3. Move to separate layer :: Choose Layer > New > Layer via cut. Rename layer as “Star” 
4. Selection: Square :: Use Polygonal Lasso for the golden square. Subtract the hole from selection.  
5. Move to separate layer :: Move to new layer using Layer via cut. Rename layer as “Square”. 
6. Selection: polygon :: Use Polygonal Lasso for the brown polygon. 
7. Move to separate layer ::  Move to new layer using layer via cut. Rename layer as “Polygon”. 
8. Selection : curve :: Use pen tool to select the blue curve. 
9. Move to separate layer ::  Move to new layer using layer via cut. Rename layer as “Curve”. 
10.  Selection : Circle :: Use paintbrush with 9-pixel hard brush (row 1 column 4). 
11.  Foreground Colour :: Choose Black as foreground color (default) 
12. Add Quick Mask :: Click Edit in Quick Mask Mode button. Add a mask by painting inside the circle. 
13.  Move to separate layer :: Return to Standard Editing mode. Move selection to new layer. Rename layer as “Circle”. 
14.  Selection : Text  :: Use Magic Wand to select text icons. 
15. Move to separate layer ::  Move to new layer using layer via cut. Rename layer as “Text”.  
16. Layer Visibility :: Switch off the background layer. 
17.   Save File :: Save as XXX_A1_Task3.psd and XXX_A1_Task3.jpg.
